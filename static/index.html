What you write is what compiles.

Content lives in JSON files. The CMS schema is closed enums and deny_unknown_fields all the way down. Mistakes fail at parse time, not at publish time. Schema migrations are minutes — the types are the migration.

Themes are CSS variables exposed through typed Loom tokens. Switch palette by changing one attribute on <html>. Thirteen named themes ship today: light, dark, dark-AMOLED, hc-dark, hc-light, sepia, warm, ocean, forest, violet, rose, auto, and a per-tenant override slot.

Type-safe authoring

Your content is a typed value. Not a template string.

Closed-enum CmsSection variants. deny_unknown_fields on every struct. Validators that fire when the JSON parses, not when the page ships.
