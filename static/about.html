Principles

What we won't compromise on.

Discipline first

Tests + docs + audit on every commit. Codified in scripts/discipline-check.sh --strict.

Typed contracts at every boundary

If the types compile, the system doesn't break at runtime.

Reproducible builds

Same input → same hash. Independently verifiable; not a trust-the-vendor claim.

Privacy by default

Self-host everything. No third-party fonts, no analytics, no telemetry, no CDNs.
